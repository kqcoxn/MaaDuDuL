## 下载前须知

**对于下列情况，MDDL 不提供支持**：

- win7 及更早版本的 windows 系统
- 各类定制系统（如 XOS 等非官方系统）与微 PE 系统（辅助安装环境）
- HarmonyOS 系统、移动设备等
- 国际服

**对于以下情况，MDDL 可能经常出现卡顿、失败等情况**：

- 网络环境波动较大或电脑硬件配置较差（经常出现游戏内微卡顿）
- 当前同时运行大量模拟器或其他高算力需求的游戏、生产力工具等
- 游戏处于开荒期，Box/事件解锁不满足 MDDL 逻辑需求（MDDL 只是帮忙完成重复日常，账号新内容无法跳过）
- 其他神秘设备原因

> 如果您不确定是否处于上述某一种情况，可以在群内请教群友。

## 下载 MDDL 压缩包

您可以通过多种方式可以下载 MDDL，**选择一种即可**。

::: tip
**您无需下载列出的所有文件，而是根据上一步确定好的文件名下载对应文件即可！**
:::

### QQ 群文件

您可以在 [MDDL 交流群（926874985）](https://qm.qq.com/q/EGUgYodNqo)内的**群文件夹**找到 `Win-X86` 与 `Mac-Arm64` 系统的安装包

![](/images/user/QQ群文件.png)

### Github Release

您可以在 Github Release 中下载 MDDL 的任意版本。

首先，打开 [MDDL Github Release 页面](https://github.com/kqcoxn/MaaDuDuL/releases)，最顶部的即为最新版。

::: tip 可能的问题：Github 页面打不开/下载缓慢

国内对于 Github 的防火墙时有时无，且速度非常不稳定。如果条件允许，请使用网络代理或加速器访问。

如果您无法通过其他途径进入 Github 页面，请使用其他方式下载 MDDL。
:::

在 Assets 中找到适合自己系统的版本，点击文件名即可下载。（若下载缓慢请参考上述问题）

![](/images/user/GithubRelease.png)

### Mirror 酱

在 [Mirror 酱](https://mirrorchyan.com/zh/projects?rid=MDDL&os=windows&arch=x64&channel=stable&source=MDDL-docsite) 官方的帮助下， MDDL 同样也接入了 Mirror 酱的国内高速下载与更新服务。

**您可以通过 [此链接](https://mirrorchyan.com/zh/projects?rid=mddl&os=windows&arch=x64&channel=stable&source=mddl-docsite) 获取任意版本的 MDDL**。

简单来说，Mirror 酱（简称 mirrorc）是一个由开源社区维护的有偿分发平台，可以理解为一个中转站，从海外源获取一份最新版存至国内站点，当您需要时直接从国内站点下载。（实际为全球加速）

由于海内外下载、站点维护等需要，mirrorc 会产生大量开销，因此 mirrorc 的服务是有偿的，但这**并不代表 MDDL 是收费的，您完全可以通过其他方式更新并无限制使用，仅在您需要 mirrorc 时才需要自行向 mirrorc 方付费，MDDL 社区并不承担此过程中的一切责任，当出现问题时您可以联系 mirrorc 客服解决。**

## 解压 MDDL

将下载好的安装包**完整解压**到一个**独立的文件夹**中，且**路径中不含中文**（包括存放 MDDL 的文件夹）。

请勿将 MAA 解压到如 `C:\Program Files\` 等需要 UAC 权限的路径。（如果看不懂，反正能不往 C 盘放就别往 C 盘放，只有 C 盘不要往系统自带的文件夹里放，自己新建一个文件夹）

如果您不了解如何解压压缩包，可以参考：[【Bilibili】电子扫盲课之如何解压藏在视频里的压缩包简易教程](https://www.bilibili.com/video/BV1tZ421N7EV)

::: danger 注意
**无论如何，请不要在压缩软件直接打开程序！**
:::

## 安装环境依赖

在文件夹内，您应该能找到类似 `DependencySetup_依赖库安装_x.x` 文件，您需要**右键此文件，并以管理员身份运行**（双击打开时无效）：

![](/images/user/环境安装包.png)

打开后，您可能需要额外安装 `winget`，只需在提醒您输入 `Y/N` 时输入 `Y` 并**按下回车键**即可：

![](/images/user/确认安装环境.png)

之后，脚本会帮您完成所有环境依赖的安装，您无需任何操作，等待此窗口自动退出即可。
